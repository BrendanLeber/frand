CONTRIBUTING

frand uses the Gitflow Workflow. This puts all of the development work on the "develop" branch. The "master" branch only stores the offical releases.

If you would like to contribute to frand please use the following steps to check out the source:

git clone https://github.com/BrendanLeber/frand.git
git checkout -b develop origin/develop
